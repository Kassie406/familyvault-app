ready-to-paste fix that will make those “Legal Status / Automation Status” labels readable without needing your exact HTML.

One-shot CSS override (drop in your global stylesheet)
/* ===== GDPR overview readability hotfix ===== */

/* 0) Scope to the Compliance > GDPR card (use any of these wrappers you have) */
.gdpr-overview, .gdpr-card, #gdpr-compliance {}

/* 1) Never dim whole columns/sections */
.gdpr-overview .gdpr-section,
.gdpr-card .gdpr-section,
#gdpr-compliance .gdpr-section {
  opacity: 1 !important;
  filter: none !important;
}

/* 2) Section titles (e.g., “Legal Status”, “Automation Status”) */
.gdpr-overview .section-title,
.gdpr-card .section-title,
#gdpr-compliance .section-title {
  color: #0f172a !important;           /* slate-900 */
  font-weight: 600;
  letter-spacing: .01em;
}

/* 3) Row labels on the left (e.g., “Data Protection Officer”, “Privacy Policy” …) */
.gdpr-overview .item-label,
.gdpr-card .item-label,
#gdpr-compliance .item-label,
.gdpr-overview .row .label,
#gdpr-compliance .row .label {
  color: #334155 !important;           /* slate-700 on light bg */
}

/* 4) Dark theme guards if you support dark mode */
.theme-dark .gdpr-overview .section-title,
.theme-dark #gdpr-compliance .section-title { color: #f3f4f6 !important; } /* gray-100 */
.theme-dark .gdpr-overview .item-label,
.theme-dark #gdpr-compliance .item-label     { color: #cbd5e1 !important; } /* slate-300 */

/* 5) If you need a muted/disabled look, dim ONLY badges/icons, not text */
.gdpr-overview .gdpr-section.is-muted .badge,
#gdpr-compliance .gdpr-section[aria-disabled="true"] .badge {
  opacity: .6;
}
.gdpr-overview .gdpr-section.is-muted .item-label,
#gdpr-compliance .gdpr-section[aria-disabled="true"] .item-label {
  opacity: 1 !important;
}

/* 6) Safety: undo any accidental transparent text */
.gdpr-overview .item-label,
#gdpr-compliance .item-label,
.gdpr-overview .section-title,
#gdpr-compliance .section-title {
  -webkit-text-fill-color: initial !important;
  mix-blend-mode: normal !important;
}


This does three key things:

Removes any opacity/filter on the entire section (the root cause of “only visible when highlighted”).

Forces titles to slate-900 and labels to slate-700 (WCAG-compliant on light).

Keeps the “muted” feel by dimming badges/icons only, not the text.
