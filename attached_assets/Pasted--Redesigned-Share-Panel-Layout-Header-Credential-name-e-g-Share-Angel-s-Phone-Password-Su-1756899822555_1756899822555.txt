🔑 Redesigned Share Panel Layout

Header

Credential name (e.g., “Share Angel’s Phone Password”)

Subtext: “Control who can view or manage this credential.”

Section 1: Shareable Link

Switch toggle: Enable shareable link

If enabled → show generated secure link with “Copy” + “Regenerate link” buttons.

Section 2: Share with Family Members

Dropdown: Select family member → shows list of members in your portal (with avatar & email).

Permission dropdown beside each member: None | View | Edit | Owner.

“+ Add another member” button to attach multiple.

Section 3: Send Link Directly

Input field with tabs for delivery type:

✉️ Email → enter one or more email addresses.

📱 Text (SMS) → enter phone numbers.

Buttons: Send Email / Send SMS (with confirmation toast).

Section 4: Actions

Update Sharing (gold button)

Cancel (text button)

✨ Example UI Flow
Share Angel’s Phone Password
-------------------------------------------------
[ ] Enable shareable link
https://familycircle.app/share/abc123
[Copy] [Regenerate]

Share with Family Members
-------------------------------------------------
+ Select member... [Dropdown: Kassandra, Dad, Mom, Family Group]
   • Kassandra        [View ▼]
   • Dad              [Edit ▼]

+ Add another member

Send Link Directly
-------------------------------------------------
[ Email | SMS ]  
Enter address/phone:  [________________________]
[Send]

-------------------------------------------------
[ Update Sharing ]   [ Cancel ]
