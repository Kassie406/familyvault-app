he (Neurodiversity) page looks great content-wise, but its styling drifts from the brand system the other pages use. Here’s a tight punch-list to make it visually consistent with the rest of the site (gold accents on dark, rounded cards, uniform chips, etc.), plus drop-in CSS/Tailwind tweaks you can paste into Replit.

Global “theme consistency” fixes (apply site-wide)

Use one palette via CSS variables

:root{
  --bg: #0b0b0b;            /* page background */
  --surface: #111;          /* cards/sections */
  --text: #e9e9e9;          /* body */
  --muted: #b3b3b3;         /* secondary text */
  --brand: #f4c542;         /* gold */
  --brand-600:#d4aa2e;      /* hover/pressed */
  --accent: #ffffff14;      /* subtle divider/surface */
  --success:#44c77d; --danger:#ff6b6b; /* keep if needed */
  --ring: #f4c54255;        /* focus ring */
}
.btn-brand{ background:var(--brand); color:#111 }
.btn-brand:hover{ background:var(--brand-600) }
.chip, .badge{ background:#ffffff12; color:var(--muted); border:1px solid #ffffff1f }
.icon-ring{ color:var(--brand); background:#f4c5421a; }
.card{ background:var(--surface); border:1px solid #ffffff12; border-radius:18px }
.section{ background:transparent; }
.section.alt{ background:#0e0e0e; }
.halo{ box-shadow:0 0 0 6px var(--ring); }

CTA buttons

Icon chips
All circular icon marks should use the “icon-ring” style (gold glyph, faint gold halo). Remove any blue/teal fills.

Testimonials
Use the same card style everywhere (.card + small gold stars + muted name line). Remove pink backgrounds that only appear on Neurodiversity.

Section spacing
Ensure uniform padding (py-16 md:py-24) between sections on all pages to match Child/Elderly/Married.
