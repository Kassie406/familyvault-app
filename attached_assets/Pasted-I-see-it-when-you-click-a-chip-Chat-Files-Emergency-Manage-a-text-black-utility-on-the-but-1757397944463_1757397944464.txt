I see it: when you click a chip (Chat / Files / Emergency / Manage), a text-black utility on the button wins in the active state, so the label goes dark on your black header. Let’s force readable text for the active chip and keep your gold-on-hover.

Paste this block LAST in your CSS (scoped to the Quick Access bar)
/* Base text for all chips (keep like the others) */
#quick-access-modal button[data-testid^="filter-"],
dialog[aria-label="Quick Access"] button[data-testid^="filter-"]{
  --tw-text-opacity: .7;
  color: rgba(255,255,255,var(--tw-text-opacity));
}

/* GOLD on hover (no bg/border changes) */
#quick-access-modal button[data-testid^="filter-"]:hover,
#quick-access-modal button[data-testid^="filter-"]:focus:hover,
dialog[aria-label="Quick Access"] button[data-testid^="filter-"]:hover,
dialog[aria-label="Quick Access"] button[data-testid^="filter-"]:focus:hover{
  --tw-text-opacity: 1 !important;
  color: #c5a000 !important;
}

/* FIX: active/selected chip must stay visible (override text-black) */
#quick-access-modal button[data-testid^="filter-"][data-state="active"],
#quick-access-modal button[data-testid^="filter-"][aria-pressed="true"],
dialog[aria-label="Quick Access"] button[data-testid^="filter-"][data-state="active"],
dialog[aria-label="Quick Access"] button[data-testid^="filter-"][aria-pressed="true"]{
  --tw-text-opacity: .9 !important;                 /* readable on dark bg */
  color: rgba(255,255,255,var(--tw-text-opacity)) !important;  /* or #c5a000 if you prefer gold */
}

/* Icons follow the label color in every state */
#quick-access-modal button[data-testid^="filter-"] *,
dialog[aria-label="Quick Access"] button[data-testid^="filter-"] *{
  color: currentColor !important;
  stroke: currentColor !important;
  fill: currentColor !important;
}


If you want the active chip to be gold instead of white, change the active block’s color: line to:

color: #c5a000 !important;


This overrides any text-black coming from the component classes, so the text stays clearly visible when a tab is selected, and you still get gold on hover.
