your admin/profile menu is getting crushed: items overlap, the hover goes neon/too dark, and it’s probably being clipped by a parent with overflow:hidden or a transform. Here’s a clean, accessible Admin Menu you can drop in that fixes all of that.

What’s going wrong (quick)

Parent container likely has overflow: hidden or transform: translateZ(0), so the menu gets clipped.

Menu items are stacking because of missing line-height and display rules.

Hover/active styles are too intense (and unreadable on dark).

z-index is too low.

Fix (drop-in)
HTML (portal the menu under <body>)
<!-- Trigger (avatar or name) -->
<button id="admin-menu-trigger" class="admin-trigger" aria-haspopup="menu" aria-expanded="false">
  <span class="avatar">A</span>
</button>

<!-- Menu PORTAL (appended near end of <body>) -->
<nav id="admin-menu" class="admin-menu" role="menu" aria-label="Admin menu" hidden>
  <div class="adm-header">
    <div class="adm-name">admin</div>
    <div class="adm-sub">admin@familycirclesecure.com</div>
  </div>
  <button class="adm-item" role="menuitem" data-nav="/security">
    <span>Security</span>
    <small>Audit & monitoring</small>
  </button>
  <button class="adm-item" role="menuitem" data-nav="/settings">
    <span>Settings</span>
  </button>

  <div class="adm-sep" role="separator"></div>

  <button class="adm-item danger" role="menuitem" data-action="logout">
    <span>Log out</span>
  </button>
</nav>

JS (positioning, keyboard access, outside-click)
<script>
(function(){
  const trig = document.getElementById('admin-menu-trigger');
  const menu = document.getElementById('admin-menu');

  function openMenu(){
    // Position below trigger
    const r = trig.getBoundingClientRect();
    const gap = 8;
    menu.style.left = Math.min(window.innerWidth - menu.offsetWidth - 8, r.left) + 'px';
    menu.style.top  = (r.bottom + gap) + 'px';
    menu.hidden = false;
    trig.setAttribute('aria-expanded','true');
    // focus first item
    const first = menu.querySelector('.adm-item'); first && first.focus();
  }
  function closeMenu(){
    menu.hidden = true;
    trig.setAttribute('aria-expanded','false');
  }

  trig.addEventListener('click', (e)=>{
    e.stopPropagation();
    menu.hidden ? openMenu() : closeMenu();
  });

  // Keyboard nav
  menu.addEventListener('keydown', (e)=>{
    const items = [...menu.querySelectorAll('.adm-item')];
    const i = items.indexOf(document.activeElement);
    if (e.key === 'Escape'){ closeMenu(); trig.focus(); }
    if (e.key === 'ArrowDown'){ e.preventDefault(); (items[i+1]||items[0]).focus(); }
    if (e.key === 'ArrowUp'){ e.preventDefault(); (items[i-1]||items[items.length-1]).focus(); }
    if (e.key === 'Tab'){ closeMenu(); }
  });

  // Outside click to close
  document.addEventListener('click', (e)=>{
    if (!menu.hidden && !menu.contains(e.target) && e.target !== trig) closeMenu();
  });
  window.addEventListener('resize', ()=>{ if(!menu.hidden) openMenu(); });
  window.addEventListener('scroll', ()=>{ if(!menu.hidden) openMenu(); }, true);

  // Actions
  menu.addEventListener('click', (e)=>{
    const btn = e.target.closest('.adm-item'); if(!btn) return;
    if (btn.dataset.nav){ window.location.assign(btn.dataset.nav); }
    if (btn.dataset.action === 'logout'){ fetch('/api/auth/logout',{method:'POST'}).then(()=>location.assign('/login')); }
    closeMenu();
  });
})();
</script>

If it still overlaps/clips

Check these on any ancestor of the old menu:

overflow: hidden → remove or move the menu to body (we already portal’d it).

transform on a parent (including transform: translateZ(0)) creates a new stacking context; the portal/fixed menu avoids that.

Low z-index—we set 10040. If you have modals at 9999+, bump to 12000.

Optional: keep two-line labels tidy

If you prefer single-line items, add:

.adm-item{flex-direction:row; align-items:center; gap:8px}
.adm-item small{display:none}


This will give you a crisp, accessible admin dropdown that doesn’t smush items or go pitch black on hover. If your trigger element has a different id/class, tell me and I’ll align the selectors.
