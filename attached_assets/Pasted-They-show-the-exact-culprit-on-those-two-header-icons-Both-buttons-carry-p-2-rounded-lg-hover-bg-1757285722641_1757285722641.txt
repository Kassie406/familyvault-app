Header code cleanup (remove Tailwind white hover)
Open client/src/components/family/Header.tsx and for the two buttons with:

data-testid="button-quick-access"

data-testid="button-toggle-layout"

…remove any hover:bg-white, hover:bg-white/5, or hover:bg-white/10 from their className. Do not add a white background utility back.

Add targeted CSS (no global side-effects)
Create (or update) client/src/styles/header.css to include:

/* ===== Header icon hover fix (precise, no collateral) ===== */

/* Base: keep transparent surface and kill white utilities/rings */
button[data-testid="button-quick-access"],
button[data-testid="button-toggle-layout"] {
  background-color: transparent !important;
  --tw-bg-opacity: 0 !important;
  --tw-ring-offset-shadow: 0 0 #0000 !important;
  --tw-ring-shadow: 0 0 #0000 !important;
  box-shadow: none !important;
  outline: 0 !important;
}

/* Hover: very light gold tint (instead of white pill) */
button[data-testid="button-quick-access"]:hover,
button[data-testid="button-toggle-layout"]:hover {
  /* light gold = rgba(212,175,55,0.08) */
  background-color: rgba(212, 175, 55, 0.08) !important;
}

Ensure header.css is imported once (e.g., in client/src/main.tsx or your global style barrel):
