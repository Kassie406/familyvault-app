Rapid hot-fix (scoped CSS + 2 small JSX tweaks)

Add a tiny scoped stylesheet

Create src/styles/action-center.css:

/* Kill any light background from wrappers (DaisyUI card, etc.) */
#action-center-shell,
#action-center-shell *:where(.card, .card-body) {
  background: transparent !important;
}

/* Force our dark gradient + no white hover, with gold glow */
#action-center-card {
  position: relative;
  border-radius: 16px; /* rounded-2xl */
  border: 1px solid rgba(255,255,255,0.08);
  background: linear-gradient(180deg, #0b0b0e 0%, #0e0e14 100%);
  box-shadow: 0 10px 28px rgba(0,0,0,.45);
  transition: box-shadow .2s ease, border-color .2s ease;
  overflow: hidden; isolation: isolate;
}

/* Gold glow on hover — no bg change */
#action-center-card:hover {
  border-color: rgba(212,175,55,0.40); /* gold */
  box-shadow:
    0 0 0 1px rgba(212,175,55,.35),
    0 18px 50px rgba(212,175,55,.12);
}

/* Keep links from turning underlined/blue here */
#action-center-card a {
  text-decoration: none;
  color: inherit;
}
#action-center-card a:hover {
  text-decoration: underline; /* optional */
}


Wrap the component + apply the id

In your Action Center component (e.g., src/components/family/ActionCenterCard.tsx):

import "@/styles/action-center.css";
// If you have a shared Card, you can still use it inside.

export function ActionCenterCard() {
  return (
    <div id="action-center-shell" className="bg-transparent p-0">
      <div id="action-center-card" className="p-4">
        {/* existing Action Center content */}
      </div>
    </div>
  );
}


This forces the dark gradient + gold glow and blocks any white backgrounds from parents. It also stops the blue/underline link look inside this card.
